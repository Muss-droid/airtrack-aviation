# Airtrack Aviation Booking Website - MVP Todo

## Overview
Build a professional aviation booking website for Airtrack Aviation (South Sudan) with online booking for cargo and passenger transfers.

## Core Files to Create/Modify (Max 8 files):

1. **src/pages/Index.tsx** - Main homepage with hero section, services overview, and booking CTAs
2. **src/components/BookingForm.tsx** - Unified booking form for both cargo and passenger bookings
3. **src/components/ServicesSection.tsx** - Display aviation services with images
4. **src/components/Header.tsx** - Navigation with logo and menu
5. **src/components/Footer.tsx** - Contact info and company details
6. **src/components/HeroSection.tsx** - Main hero banner with background image
7. **src/App.tsx** - Update routing and layout
8. **index.html** - Update title and meta tags

## Key Features:
- Professional hero section with Airtrack branding
- Service showcase (Passenger Charter, Cargo, Medical)
- Online booking form for both services
- Responsive design with aviation theme
- Contact information and company details
- Image gallery of services

## Design Approach:
- Use blue/orange color scheme from logo
- Professional aviation industry styling
- Clean, modern interface
- Mobile-responsive layout
- High-quality images from uploads

## Technical Implementation:
- React + TypeScript + Tailwind CSS
- Shadcn-ui components for forms and UI
- Local state management for booking data
- Form validation and user feedback
