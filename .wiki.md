# Project Summary
The Airtrack Aviation project aims to develop a modern web application for Airtrack Aviation based in South Sudan, providing online booking services for both cargo and passenger transfers. The application will feature a professional design, leveraging uploaded imagery to enhance user experience and promote the company's aviation services.

# Project Module Description
The application consists of several functional modules, including:
- **Header**: Contains navigation and branding.
- **Hero Section**: Displays the main branding and calls to action.
- **Services Section**: Showcases the various services offered by Airtrack Aviation.
- **Booking Form**: Allows users to book passenger or cargo services.
- **Footer**: Provides contact information and links to social media.

# File Description Inventory
- **README.md**: Project documentation and overview.
- **components.json**: Configuration for components.
- **eslint.config.js**: ESLint configuration for code quality.
- **index.html**: Main HTML file for the web application.
- **package.json**: Project dependencies and scripts.
- **postcss.config.js**: Configuration for PostCSS.
- **public/**: Directory for public assets including images and favicon.
- **src/**: Source code for the application, including components, pages, and styles.
- **uploads/**: Directory containing uploaded images used in the application.

# Technology Stack
- **React**: For building the user interface.
- **TypeScript**: For type safety and better development experience.
- **Tailwind CSS**: For styling the application.
- **Shadcn-ui**: UI components library for building user interfaces.
- **date-fns**: For date handling.

# Usage
1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Run linting checks:
   ```bash
   pnpm run lint
   ```
