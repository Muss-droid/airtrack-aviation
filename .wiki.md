# Project Summary
The AIRTRACK AVIATION project is a modern web application designed for AIRTRACK AVIATION in South Sudan, providing online booking services for cargo and passenger transfers. The platform enhances user experience through a professional interface, facilitating seamless customer interactions and effectively promoting the company's services. Recent updates include the integration of a new logo to strengthen brand identity.

# Project Module Description
The application consists of several functional modules:
- **Header**: Navigation and branding with booking functionality, now featuring the updated logo.
- **Hero Section**: Main branding and calls to action.
- **Services Section**: Overview of services offered.
- **Booking Form**: Interface for booking passenger or cargo services, with enhanced validation and feedback.
- **Footer**: Contains updated contact information, social media links, and the new logo.

# Technology Stack
- **React**: For building the user interface.
- **TypeScript**: For type safety and better development experience.
- **Tailwind CSS**: For styling the application.
- **Shadcn-ui**: UI components library for building user interfaces.
- **date-fns**: For date handling.
- **Sonner**: For toast notifications.

# Usage
1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Run linting checks:
   ```bash
   pnpm run lint
   ```
