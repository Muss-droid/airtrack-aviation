# Project Summary
The AIRTRACK AVIATION project is an innovative web application aimed at transforming aviation services in South Sudan. It offers a seamless online platform for booking cargo and passenger transfers, featuring a professional user interface and efficient customer engagement. The application includes specialized services such as emergency medical transport and consultation, ensuring an exceptional user experience that reflects the company’s commitment to operational excellence.

# Project Module Description
The application comprises several functional modules:
- **Header**: Navigation and branding with booking functionality, featuring the updated logo.
- **Hero Section**: Main branding area with an updated background image showcasing aircraft.
- **Services Section**: Overview of services offered, including a "Why Choose AIRTRACK AVIATION?" section with an engaging background image.
- **Gallery Carousel**: Interactive carousel displaying operational photos, enhancing visual engagement.
- **Booking Form**: Interface for booking passenger or cargo services, with improved validation and feedback.
- **Footer**: Contains updated contact information, social media links, and the new logo.

# Technology Stack
- **React**: For building the user interface.
- **TypeScript**: For type safety and better development experience.
- **Tailwind CSS**: For styling the application.
- **Shadcn-ui**: UI components library for building user interfaces.
- **date-fns**: For date handling.
- **Sonner**: For toast notifications.

# Usage
1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Run linting checks:
   ```bash
   pnpm run lint
   ```
