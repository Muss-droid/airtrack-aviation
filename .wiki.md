# Project Summary
The AIRTRACK AVIATION project is a modern web application designed to enhance aviation services in South Sudan. It provides a seamless online booking experience for cargo and passenger transfers, emphasizing a professional user interface and efficient customer interactions. The application showcases the company's offerings, including emergency medical transport and consultation services, while ensuring an engaging user experience.

# Project Module Description
The application consists of several functional modules:
- **Header**: Navigation and branding with booking functionality, featuring the updated logo.
- **Hero Section**: Main branding area with an updated background image showcasing aircraft.
- **Services Section**: Overview of services offered, including a "Why Choose AIRTRACK AVIATION?" section with an engaging background image.
- **Booking Form**: Interface for booking passenger or cargo services, with enhanced validation and feedback.
- **Footer**: Contains updated contact information, social media links, and the new logo.

# Technology Stack
- **React**: For building the user interface.
- **TypeScript**: For type safety and better development experience.
- **Tailwind CSS**: For styling the application.
- **Shadcn-ui**: UI components library for building user interfaces.
- **date-fns**: For date handling.
- **Sonner**: For toast notifications.

# Usage
1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Run linting checks:
   ```bash
   pnpm run lint
   ```
